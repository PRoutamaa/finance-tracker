# Finance Tracker

## Idea

The Finance tracker is a simple app to enter and track incomes and spendings, balance between them, follow statistics 
and check history of all transactions entered.
This app consists of 4 pages: main/dashboard, new income, statistics and history, 
and a navigation bar on the left (browser user) or on top (mobile user).

### Main/dashboard
This page shows the total spendings and incomes as well as overall balance.

### New income
This page is for entering new income or spending:
    -Required fields:
        -Title
        -Amount in euros
        -Date, current date as default
        -Checkbox for income

### Statistics
This page is for viewing statistics of the transactions from the past 30 days

### History
This page shows all the transactions made as a list

## How to use

To run the app locally, clone the repo from github: https://github.com/PRoutamaa/finance-tracker

run the app from the root of the project with: flutter run -t src/main.dart -d chrome
